# 🎓 LearnHub Online Course Platform (FastAPI)

A fully functional backend system built using **FastAPI** that simulates an online course platform with course management, enrollments, wishlist, filtering, search, sorting, and pagination.

---

## 🚀 Features

### 📚 Course Management
- Get all courses with total and available seats
- Get course by title
- Course summary (free courses, most expensive, category breakdown)
- Add new course
- Update course (price / seats)
- Delete course (with validation)

---

### 🧑‍🎓 Enrollments
- Enroll in a course
- Apply discounts (early-bird + coupons)
- Gift enrollment support
- View all enrollments
- Search, sort, and paginate enrollments

---

### ❤️ Wishlist System
- Add course to wishlist
- Remove course from wishlist
- View wishlist with total value
- Enroll all wishlist courses

---

### 🔍 Advanced Features
- Filter courses (category, level, price, seats)
- Search courses (title, instructor, category)
- Sort courses (price, title, seats)
- Pagination for courses and enrollments
- Combined browsing (search + filter + sort + paginate)

---

## 🛠️ Tech Stack

- Python
- FastAPI
- Pydantic
- Uvicorn
